LINE設定

LINE公式アカウントから患者・管理者へメッセージを送信するための認証キーです。 LINE Developers の対象チャネル → Messaging API → チャネルアクセストークンで取得します。

緊急回答が届いたときに通知を受け取る医療者のLINE user_idです。 複数人へ通知する場合はカンマ区切りで入力します。 管理者本人がLINE公式アカウントへメッセージを送ると、初期セットアップ画面の候補に表示されます。
Google設定
Google Sheets連携の手順
  1. Google CloudでサービスアカウントJSONを作成します。
  2. この画面または初期セットアップ画面でJSONファイルをアップロードします。
  3. 下に表示される「共有先メールアドレス」をコピーします。
  4. 使いたいGoogleスプレッドシートを開き、右上の「共有」を押します。
  5. 共有先メールアドレスを貼り付け、権限を「編集者」にして共有します。
  6. スプレッドシートURLの /d/ と /edit の間をスプレッドシートIDとして入力します。

Google Sheets 右上の「共有」から、このメールアドレスを編集者として追加してください。 403エラーが出る場合は、ほぼこの共有設定が原因です。

患者情報、安否回答、未登録ユーザー、モード設定を保存するGoogleスプレッドシートのIDです。 Google Sheets のURL内の /d/ と /edit の間にある文字列を入力します。 例: https://docs.google.com/spreadsheets/d/ここがID/edit
