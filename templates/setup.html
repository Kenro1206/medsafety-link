LINE設定

■ これは何？
→ LINE公式アカウントがメッセージを送信するための鍵です。

■ 何に使う？
→ 患者へ安否確認メッセージを送るために使います。

■ どこで取得？
→ LINE Developers → 対象チャネル → Messaging API → チャネルアクセストークン

■ 注意
→ できれば長期アクセストークンを使用してください。
Google設定

■ これは何？
→ Google Sheets にアクセスするための認証ファイルです。

■ 何に使う？
→ 患者情報、安否回答、ログなどをスプレッドシートに保存するために使います。

■ どこで取得？
→ Google Cloud → サービスアカウント → JSONキーを作成してダウンロードします。

■ 現在の設定値
→ {{ settings.google.service_account_file }}

■ 注意
→ このファイルは認証情報なので、外部に漏れないように管理してください。
{% if not settings.google.service_account_file %}
Google連携には JSON ファイルのアップロードが必要です。
{% endif %}

■ これは何？
→ データ保存先となる Google スプレッドシートの ID です。

■ 何に使う？
→ 患者情報、安否回答、モード設定などの保存先になります。

■ どこで確認？
→ Google Sheets の URL の /d/ の後ろの文字列です。

■ 例
→ https://docs.google.com/spreadsheets/d/ここがID/edit
